# Task Advancer

Инструмент автоматического продвижения заданий DMC по статусам жизненного
цикла. Доводит задание до нужного этапа без ручного прохождения через
Manager / Monitor / Print Station и без реального оборудования линии.

Третье приложение линейки инструментов (вместе с **KMTasker** — создание
заданий и **LineEmulator** — эмуляция оборудования), в едином стиле.

## Функциональность

- **Продвижение по статусам** — автоматическое доведение задания до выбранной цели: «Требуется отправить отчёт о нанесении КМ», «Можно передать данные для ввода в оборот» или «Завершено»
- **Верификация КМ и агрегатов** — эмуляция результата работы линии (печать и проверка кодов камерой)
- **Сборка агрегации** — предзарезервированные КИГУ наполняются кодами, КИТУ и паллеты собираются снизу вверх тем же путём, что у настоящего Monitor; коды агрегатов в формате штатного генератора DMC
- **Ожидание готовности** — задание в подготовительном статусе (создание, резерв КМ) можно поставить в очередь: программа дождётся готовности и запустит продвижение сама
- **Автодополнение номера задания** — подсказки из БД только по заданиям, готовым к продвижению; вставка из буфера на любой раскладке
- **Параллельное отслеживание** — каждое задание идёт в своём потоке; ошибка одного не влияет на остальные, повтор одной кнопкой
- **Защита от залипаний** — автоповтор перевода статуса, если сервер не подхватил; распознавание серверной автоотправки (сервер прошёл целевой статус сам)
- **Уведомления** — звук, всплывающее уведомление и зелёная подсветка карточки при достижении цели
- **Проверка подключений** — автоматическая диагностика БД и API при запуске

## Запуск

```
pip install -r requirements.txt
python src/main.py
```

Сборка exe: `pyinstaller --noconfirm TaskAdvancer.spec`

## Настройки

⚙️ в сайдбаре: PostgreSQL DMC (host/port/dbname/user/password), REST API v2
(host/port), интервал опроса статуса, звук уведомлений. Настройки хранятся
рядом с приложением (`db_config.json`, `api_config.json`, `app_config.json`),
список заданий переживает перезапуск (`tasks.json`), подробный лог —
`logs/taskadvancer_log.txt`.
